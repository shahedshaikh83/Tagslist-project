Common HTML Tags

Tag Name Use / Description
<html> HTML Defines the root of an HTML document
<head> Head Contains meta-information about the document
<title> Title Sets the title of the document (shown in browser tab)
<body> Body Defines the body content of the document
<h1> to <h6> Headings aDefines headings from largest (h1) to smallest (h6)
<p> Paragraph Defines a paragraph
<a> Anchor Creates a hyperlink
<img> Image Embeds an image
<ul> Unordered List Defines a bullet-point list
<ol> Ordered List Defines a numbered list
<li> List Item Defines a list item in ul or ol
<table> Table Defines a table
<tr> Table Row Defines a row in a table
<td> Table Data Defines a cell in a table row
<th> Table Header Defines a header cell in a table
<form> Form Defines an HTML form for user input
<input> Input Defines an input field
<button> Button Defines a clickable button
<div> Division Defines a division or section
<span> Span Defines a section of text inline
<br> Line Break Inserts a single line break
<hr> Horizontal Rule Inserts a horizontal line
<meta> Meta Defines metadata about the HTML document
<link> Link Links external resources like CSS
<script> Script Embeds or references JavaScript code
<video> video Embeds videos of youtube/browder and local device
